const 指针

const在*左边，是指指针所指的对象是不可以改变的；在*右边，则是指指针不能指向另外一个对象。

[codesyntax lang="cpp"]

int b = 10;
const int *a = &b;//or int const *a = &b;
*a = 11;//error

int d = 20;
int * const c = &b;
c = &d;//error

[/codesyntax]
